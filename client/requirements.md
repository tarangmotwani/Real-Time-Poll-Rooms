## Packages
framer-motion | Smooth animations for poll bars and page transitions
react-use-websocket | Robust WebSocket handling for real-time updates
recharts | Beautiful data visualization for poll results
nanoid | Generating unique voter tokens on the client side
clsx | Class name utility
tailwind-merge | Class name utility

## Notes
WebSocket connects to window.location.host
Voter token is stored in localStorage ('poll_voter_token')
Poll creation redirects to /poll/:id
